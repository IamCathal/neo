FAQ

What is this project?

Neo is an NUIG Computer Science and Information Technology final year project built to analyse friend networks on the Steam platform

How do I get a steamID64 for a Steam account?

Each Steam account has a ID associated with it thats used behind the scenes to uniquely identify a given account. The easiest way to get the associated steamID64 for a given account is through steamIDFinder

What is Neo built with?

Microservices

  • Frontend: Go with vanilla Javascript and Bootstrap
  • Crawler: Go
  • Datastore: Go

Infrastructure

  • Messaging: RabbitMQ
  • Data Persistence: MongoDB and PostgreSQL
  • Logging: ELK stack (Elasticsearch, Logstash and Kibana)
  • Metrics: Grafan and InfluxDB
  • Uptime Monitoring: Uptime Kuma

Is this project FOSS?

Yes, Neo is is licensed under MIT and the source code is available on Github
